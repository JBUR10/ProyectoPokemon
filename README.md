# Galería Pokémon

## Descripción

Aplicación web desarrollada con HTML, CSS y JavaScript que consume la API pública PokeAPI para mostrar información de diferentes Pokémon.

## API utilizada

PokeAPI

https://pokeapi.co/

## Endpoints consumidos

Listado de Pokémon

https://pokeapi.co/api/v2/pokemon?limit=12&offset=0

Detalle de un Pokémon

https://pokeapi.co/api/v2/pokemon/{id}

## Funcionalidades

- Mostrar Pokémon
- Buscar Pokémon
- Cargar más Pokémon
- Manejo de errores
- Diseño responsive

## Tecnologías

- HTML
- CSS
- JavaScript
- Fetch API

## Ejecución

1. Descargar el proyecto.
2. Abrir el archivo index.html en un navegador.
3. Tener conexión a Internet.
